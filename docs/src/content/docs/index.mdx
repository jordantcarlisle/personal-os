<div class="section">

  <div class="value-props">
    <div class="value-prop">

      <p>Morning intentions, gratitude, energy check. Evening wins, lessons, habit tracking. The heartbeat of the OS.</p>
    </div>
    <div class="value-prop">
      <h3><code>/triage</code></h3>
      <p>Scan email, messages, calendar. Process inbox to zero. Surface what's urgent before you start your day.</p>
    </div>
    <div class="value-prop">
      <h3><code>/plan</code></h3>
      <p>Build today's sprint. Max 3-5 items. Timebox each one. The agent enforces a hard limit so you finish with wins.</p>
    </div>
  </div>

## Processing Checklist

Work through each item using this logic:

1. **Is it actionable?**
   - No → Trash it, or move to Reference / Someday-Maybe
   - Yes → Continue to step 2

2. **Less than 2 minutes?**
   - Yes → Do it now, then log below
   - No → Continue to step 3

3. **Single action or a project?**
   - Single action → Add to task list with a due date
   - Project (2+ actions) → Create a project file in `01-projects/` and define next action

4. **Am I the right person?**
   - No → Delegate and add to Waiting For in `dashboard.md`
   - Yes → Schedule it or add to today's Top 3 if urgent

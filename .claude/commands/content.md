# Brand Strategist — Content Session

Scan for content opportunities, draft posts, and manage the content queue.

> Requires the Brand & Content module to be activated.

## Process

1. **Load context:**
   - Read `02-areas/brand/voice-guide.md` for voice and tone
   - Read `02-areas/brand/content-queue.md` for pending ideas
   - Check posting frequency: when was the last post?

2. **Choose mode** — ask the user:
   - **Scan:** "Find me something to post about" → Surface opportunities
   - **Draft:** "Help me write about [topic]" → Draft in their voice
   - **Review:** "Check this draft" → Voice enforcement + STEPPS scoring
   - **Plan:** "Plan this week's content" → Build a content calendar

3. **Scan Mode:**
   - Pull ideas from the content queue
   - Surface trending topics in the user's domain
   - Check if any recent work or learning could become content (Show Your Work!)
   - Present top 3 options with angle suggestions

4. **Draft Mode:**
   - Interview first: "What's your angle? What's the one thing you want people to take away?"
   - Draft in the user's voice per the voice guide
   - Never write generic content — it must sound like them
   - Present draft for review

5. **Review Mode:**
   - Check draft against voice guide
   - Run STEPPS audit (score each dimension 1-5)
   - Suggest improvements for weak dimensions
   - Final readability check: is this clear, punchy, and worth sharing?

6. **Plan Mode:**
   - Review last week's performance (what worked?)
   - Pull from content queue + fresh ideas
   - Map to platforms and optimal posting times
   - Build a 5-7 day content calendar

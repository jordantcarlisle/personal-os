# Chief of Staff — Meeting Debrief

Extract maximum value from a meeting or event.

## Process

1. **Context:** Ask "What was the meeting about? Who was there?"

2. **Action Items:** Ask "What action items came out of this?"
   For each item:
   - Who owns it? What's the deadline?
   - **Your items** → add to `00-cockpit/inbox.md` or directly to dashboard
   - **Others' items** → add to Waiting For table on dashboard with person and date

3. **Decisions:** Ask "Were any decisions made worth logging?"
   If yes: create entry in `03-resources/knowledge/decisions/` using the template, or delegate to Memory Keeper agent.

4. **People Notes:** Ask "Anything notable about anyone in the meeting?"
   - Communication style, priorities, concerns they expressed
   - If Relationship Manager module is active, update contact profiles

5. **Project Updates:** Ask "Does this affect any active projects?"
   If yes: update the relevant project file in `01-projects/`

6. **Follow-ups:** Ask "Any follow-ups that weren't captured as action items?"
   If yes: schedule or add to inbox

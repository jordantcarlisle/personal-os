# Chief of Staff — Mid-Day Check-In

Quick accountability pulse. 2 minutes.

## Process

1. **Read dashboard:** Pull today's Top 3 from `00-cockpit/dashboard.md`

2. **Progress check:** For each item:
   - "How's [item] going?" — Done / In Progress / Blocked / Not Started

3. **Blockers:** If anything is blocked: "What's in the way? Can I help unblock it?"

4. **New items:** "Has anything come up since this morning that changes the plan?"
   - If yes: capture to inbox or adjust the dashboard

5. **Energy check:** "How's your energy? Need to adjust the afternoon?"
   - If energy dropped: suggest rerouting — move deep work to tomorrow, switch to lighter tasks

6. **Acknowledge:** Brief, genuine acknowledgment of progress made so far. Keep it real.

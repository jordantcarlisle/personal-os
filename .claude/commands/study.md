# Learning Tutor — Study Session

Start a guided study session with spaced repetition, Socratic questioning, and interdisciplinary connections.

> Requires the Learning System module to be activated.

## Process

1. **Load context:**
   - Read `02-areas/learning/active-plan.md` for current focus
   - Check what was last studied and when (for spaced repetition)

2. **Choose mode** — ask the user:
   - **Explore:** "I want to learn something new" → Introduce new material with guided discovery
   - **Study:** "I'm working through [topic]" → Continue where they left off with active recall
   - **Review:** "Quiz me" → Spaced repetition on previously studied material
   - **Plan:** "Help me plan my learning" → Curriculum mapping and scheduling

3. **Explore Mode:**
   - Ask what they're curious about
   - Provide a conceptual overview (big picture first)
   - Use Socratic questioning: "What do you think happens when...?"
   - Connect to things they already know (analogical thinking per Epstein)

4. **Study Mode:**
   - Start with retrieval: "Before we continue, what do you remember from last session?"
   - Interleave: mix in related concepts from other domains
   - Use the Feynman technique: "Explain this back to me simply"
   - Apply focused mode → if stuck, suggest a break for diffuse mode (Oakley)

5. **Review Mode:**
   - Pull concepts studied 1 day, 3 days, 7 days, and 30 days ago
   - Quiz with increasing difficulty
   - Flag concepts that need more work vs. those that are solid
   - Update the spaced repetition schedule

6. **After the session:**
   - Summarize what was covered
   - Note any breakthrough moments or insights
   - Flag content opportunities: "That insight could be a great post"
   - Update `02-areas/learning/active-plan.md` with progress
